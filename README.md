# 💰 FinanzApp – El juego del emprendedor

![FinanzApp Banner](https://img.icons8.com/fluency/96/000000/money--v1.png)

**FinanzApp** es un juego interactivo y progresivo diseñado para enseñar finanzas básicas de negocios y emprendimientos de una manera divertida y adictiva. Inspirado en juegos como Coddy, pero enfocado en llevar cuentas, calcular presupuestos y entender la salud financiera de un pequeño negocio.

> "El que quiere, puede" – Creado desde un teléfono móvil para demostrar que no se necesitan grandes equipos para hacer grandes cosas.

---

## 🚀 Características

- 🎮 **Mapa de niveles** con desbloqueo progresivo.
- 🧠 **Preguntas prácticas** de finanzas: ganancias, porcentajes, punto de equilibrio.
- ✅ **Retroalimentación inmediata** con sonidos y vibración (personalizable).
- 💡 **Consejos financieros** útiles para emprendedores.
- ⚙️ **Ajustes** para activar/desactivar sonido y vibración.
- 💾 **Guardado automático** del progreso (localStorage).
- 📱 **Diseño 100% responsive** pensado para móviles (pero funciona en desktop).
- 🔊 **Efectos de sonido** integrados (Web Audio API).
- 📳 **Vibración** en dispositivos compatibles.
